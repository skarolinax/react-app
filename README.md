# React to do app

This is a simple interactive to do application built with React.

## Usage

1. Type a task in the input field and press Enter or click Add task.

2. Mark a task as done by clicking the checkbox. Confetti will celebrate your progress!

3. Delete tasks by clicking the trash icon.

4. When all tasks are done, a modal will appear to congratulate you.

## Technologies Used

- React – front-end framework

- SCSS – styling with variables, mixins, and nesting

- JS Confetti – for fun celebration effects

- Font Awesome – icons for buttons
